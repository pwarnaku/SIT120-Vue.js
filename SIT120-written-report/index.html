Introduction

The HireME.com website which will be using as an online study helper for university students that seek help for their tuff university projects is implemented in vue.JS and Vuetify.js using Vue cli and basic html, CSS.


Vuetify.js is a package which implement the martial design for vue.js components Detailed instruction can be found at Vuetifyjs


How to set-up

Command Permissions What does
$ npm install -g vue-cli Root/Administrator Installs the Vue CLi
$ vue create "app-name" - Creates the Vue app
cd "app-name" - Navigates into the new folder created
$ npm install - Installs dependencies
$ npm run dev - Serves with hot reload at localhost:8080

Run the application

**Should have Vue cli installed.

Simply navigate into the project folder and give following commands.

  • $ npm install
  • $ npm run dev
